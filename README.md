# 🧠 Did I?

**Did I?** is a Progressive Web App (PWA) built with **Next.js** that helps users verify and remember everyday actions ... reducing uncertainty, rechecking, and anxiety.

It focuses on **confirmation and proof**, not productivity.

---

## ❓ The Problem

You’re already outside, and suddenly the thought hits you:

- *Did I lock the door?*
- *Did I turn off the microwave?*

That small doubt often turns into rechecking, anxiety, and distraction — not because we forgot, but because we’re not sure.

**Did I?** was built to solve that human problem.

---

## 🚀 Features

- **Custom task creation**  
  Add, edit, archive, and reorder your own tasks

- **Per-task photo proof**  
  Attach one or more photos under each task as confirmation

- **Persistent storage**  
  Tasks, photos, history, and settings are saved and survive refresh

- **Task confirmation history**  
  Clear timeline to replace repeated rechecking

- **Task states**  
  Confirmed, archived, and restored

- **Anxiety-handling dashboard**  
  Designed for reassurance and clarity

- **Settings panel**
  - Dark mode
  - Calm / reduced-animation mode
  - Preferences saved persistently

- **Next.js PWA support**
  - Installable
  - Offline-ready
  - Fast and reliable

- **Clean, modular architecture**
  - Components
  - Hooks
  - Services
